DEUTLI Extractor is a utility for extracting hidden metadata (generation parameters, prompts, workflows) from PNG files and saving them into a standardized text format.

1. How It Works

The utility operates locally, directly interacting with your file system without uploading anything to the network.

  • Step 1: Directory Selection. Choose the folder containing your images.
    Security Note: Since the Extractor is a web application (PWA), the browser will request system permission to view and edit files in the selected directory in order to read the images and save the text files.
  • Step 2: Scanning. The application will read the files. Please note: rendering thumbnails for folders with a large number of heavy images may take some time.
  • Step 3: Choose Operation Mode.
    Batch Export: Click the EXTRACT ALL TO .DEUT button to automatically generate companion text files for all images in the folder at once.
    Selective Operation: Use the view toggles (Grid / List) to navigate and select a specific file.
  • Step 4: Inspector (View and Export). Clicking on a specific image will display the extracted metadata in the right panel. You can review the data, copy the full original graph (Raw Workflow) to the clipboard in JSON format, or save a standalone .deut file for the selected image only.

2. Privacy and Autonomy (100% Local PWA)

All file parsing processes are executed exclusively within your browser's RAM. Your data never leaves your computer.

  • The utility is a Progressive Web App (PWA). You can install it as a standalone application using your browser's native functionality.
  • Once installed, the Extractor is cached in your system and can fully operate without an internet connection.

3. The Open .deut Standard

We adhere strictly to the Sidecar concept. The Extractor does not alter original images under any circumstances—your PNGs remain untouched. All extracted information is saved alongside the original files in a lightweight JSON format with the .deut extension.
The format specification is open and available for review and integration: Open Deut Format on GitHub.

4. Open Source

DEUTLI Extractor is an open-source tool. The entire parser code (including the recursive search algorithm for complex ComfyUI graphs) is completely transparent. You can inspect the source code and contribute to its development in our repository: DEUTLI Extractor on GitHub.
